Online HNN Workshop, Summer 2026

Welcome to our HNN workshop in collaboration with Neuroscience Gateway Portal (NSG). In this hybrid lecture-based/interactive workshop, we will begin with a didactic overview of the background and development of HNN. We will then introduce you to the graphical user interface (GUI) and application programming interface (API) of HNN through demonstrations and hands-on investigations of one of the most common non-invasively recorded brain signals: event-related potentials (ERPs). We will also cover automatic parameter optimization in the HNN GUI and parameter sweeps in the HNN Python API using simulated ERPs as an example.

Installing and Running HNN

Neuroscience Gateway Portal

For the majority of the workshop, we will be using HNN through the Neuroscience Gateway Portal (NSG). You can find instructions for using HNN on NSG at this page here.

Note that the commands used to create a zip archive that are provided in the instructions are specific to MacOS/Linux. If you are using a Windows machine, you can simply right-click on file and select the option to compress to a zip file from the context menu.

Running HNN GUI in the Cloud (Google CoLab)

For the HNN API section of the workshop, we will use interactive Google CoLab notebooks to illustrate various concepts. This method of running HNN requires a Google account, but the resources are free to use so long as you have an account.

Running HNN Locally

Altenatively, if you would prefer to run HNN locally rather than via NSG or Google Colab, please follow our local installation instructions.

If you are using MacOS or Linux, we recommend that you follow the Conda Package Installation instructions, which will set up the MPI backend for running parallel simulations, which greatly increases simulation speeds.
